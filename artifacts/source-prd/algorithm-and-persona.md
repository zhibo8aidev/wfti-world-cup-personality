# 算法与人格数据基线 - WFTI 世界杯球迷人格大鉴赏

## 1. 评分计算
1. 收集 16 题答案，每题得分为 1 / 2 / 3。
2. 8 个维度按两题聚合：
   - F1 = 题1 + 题2
   - F2 = 题3 + 题4
   - E1 = 题5 + 题6
   - E2 = 题7 + 题8
   - S1 = 题9 + 题10
   - S2 = 题11 + 题12
   - A1 = 题13 + 题14
   - A2 = 题15 + 题16
3. 等级映射：
   - 2~3 分 -> L -> 数值 1
   - 4 分 -> M -> 数值 2
   - 5~6 分 -> H -> 数值 3
4. 生成用户向量：[F1, F2, E1, E2, S1, S2, A1, A2]

## 2. 隐藏人格优先规则
- TROL / 杠精球评家：题3=A + 题12=C + 题9=A
- HERO / 四年一醒：题13=A + 题16=C + 题7=C
