# Design Spec - WFTI 世界杯球迷人格大鉴赏

## 1. 设计概览
本项目定位为 2026 世界杯期间直播吧 APP 内的 H5 互动活动。设计核心在于：**极致的沉浸式衔接、细腻的拟真触感、工业级的阅读秩序**。

- **平台**: Mobile-web (H5)
- **视觉关键词**: 连续影棚地板、微渐变、弥散投影、内高光、高对比度排版
- **配色方案**: 
    - **主背景色**: 待定（严格吸附头图底部像素色，暂定 #F5F7FA 灰白调性）
    - **品牌主色**: 直播吧蓝（建议采用 #007AFF 及其拟真微渐变变体）
    - **辅助色**: 极客蓝（用于插画与标签云）、警告红、成功绿
    - **文字色**: #1D1D1F (深黑)、#86868B (中灰)、#AEAEB2 (浅灰)

## 2. 设计系统 (Atomic Design)

### 2.1 纳米级触感系统 (Tactile Components)
- **主按钮 (Primary Button)**:
    - 圆角: 12px
    - 背景: 微渐变 (如 `linear-gradient(180deg, #3395FF 0%, #007AFF 100%)`)
    - 阴影: 大半径弥散投影 (`0 8px 24px rgba(0, 122, 255, 0.2)`)
    - 细节: 1px 半透明白色内高光描边 (顶部)
    - 状态: 默认、按下 (scale 0.96 + 阴影减弱)、禁用 (灰度 + 透明度)、加载中 (旋转进度图标)

- **选项卡片 (Choice Card)**:
    - 纯白背景，细微描边 (1px #E5E5E7)
    - 选中态: 主色描边 + 内发光
    - 间距: 12px gutter

### 2.2 排版秩序 (Typography)
- **字体栈**: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "PingFang SC", "Microsoft YaHei", sans-serif
- **层级**:
    - 人格代号: 40px, Bold, Letter-spacing: -1px
    - 页面标题: 24px, Medium
    - 正文: 16px, Regular, Line-height: 1.6
    - 标签/注释: 12px, Regular, Letter-spacing: 0.5px

## 3. 页面详细设计

### 3.1 活动落地页 (Landing Page)
- **顶部**: 满宽头图 (占位符)，高度约 40vh。
- **背景**: 底部颜色与头图无缝衔接。
- **内容**: 
    - Slogan: 居中大字，体现力量感。
    - 说明文案: 模块化卡片封装。
- **底部**: 固定悬浮区域，包含“开始鉴赏”按钮。

### 3.2 答题页 (Question Page)
- **进度条**: 顶部极简线形，百分比跟随。
- **题干**: 位于页面中上部，大字呈现。
- **选项**: 垂直排列，卡片形式，具备触感反馈。
- **切题动画**: 水平平滑滑入，保持流程的丝滑感。

### 3.3 结果计算过渡页 (Transition Page)
- **核心**: 旋转的足球 DNA 序列或类似动效。
- **文案**: “正在扫描你的球迷基因...”、“正在匹配 2026 赛事向量...”。

### 3.4 结果页 (Result Page)
- **人格卡片**: 采用类似名片的圆角卡片布局。
- **插画区**: 扁平化极客蓝风格插画 (占位)。
- **雷达图**: 8 维数据可视化，展示性格分布。
- **关系映射**: “最佳拍档”与“命定天敌”采用双头像对撞设计。

### 3.5 分享海报 (Poster)
- **布局**: 纵向长图 (750x1334)。
- **包含**: 顶部人格头衔、中部雷达图、底部二维码、直播吧 Logo。

## 4. 异常状态处理
- **活动未开始**: 全屏插画 + “距离开启还有 X 天”倒计时。
- **活动已结束**: TOP3 热门人格排行榜单，引导查看往期精彩。
- **网络异常**: 断网插画 + 显眼的主色重试按钮。

## 5. 待执行项 (Pending Items)
- [ ] **吸色校准**: 待头图原图补齐后，需对页面背景色 `#F5F7FA` 进行像素级校准。
- [ ] **插画绘制**: 目前使用几何占位，需补充 18 个人格对应的扁平化插画。
- [ ] **球星映射**: 需产品补齐每个结果对应的球星对照表。

---
*注：本设计方案遵循工业级阅读秩序与拟真触感原则，旨在提升用户的参与感与分享欲。*
