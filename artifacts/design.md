# Design Spec - WFTI 世界杯球迷人格大鉴赏 (v2 Refined)

## 1. 设计概览
本项目定位为 2026 世界杯期间直播吧 APP 内的 H5 互动活动。设计核心在于：**极致的沉浸式衔接、细腻的拟真触感、工业级的阅读秩序**。

- **平台**: Mobile-web (H5)
- **视觉关键词**: 连续影棚地板、微渐变、弥散投影、内高光、高对比度排版
- **配色方案**: 
    - **主背景色**: `#2586EB` (首屏衔接色) -> `#F0F7FF` (正文背景渐变过渡)
    - **品牌主色**: `#2E7DEB` (基于头图蓝色的品牌延展)
    - **按钮梯度**: `linear-gradient(180deg, #5FA2FF 0%, #2E7DEB 100%)`
    - **辅助色**: `#EBF5FF` (浅蓝背景)、`#FF4D4F` (警告红)、`#52C41A` (成功绿)
    - **文字色**: `#1D1D1F` (深黑)、`#4A4A4C` (次级深灰，提升对比度)、`#86868B` (中灰)

## 2. 设计系统 (Atomic Design)

### 2.1 纳米级触感系统 (Tactile Components)
- **主按钮 (Primary Button)**:
    - 圆角: 12px
    - 背景: 微渐变 `linear-gradient(180deg, #5FA2FF 0%, #2E7DEB 100%)`
    - 阴影: 0 8px 20px `rgba(46, 125, 235, 0.25)`
    - 细节: 1px 半透明白色内高光描边 (顶部)
    - 状态: 默认、按下 (scale 0.96 + 阴影减弱)、禁用 (灰度 + 透明度)、加载中 (旋转进度图标)

- **选项卡片 (Choice Card)**:
    - 纯白背景，细微描边 (1px #D1E9FF)
    - 选中态: `#2E7DEB` 描边 + 0 0 12px `rgba(46, 125, 235, 0.15)`
    - 间距: 12px gutter

### 2.2 排版秩序 (Typography)
- **字体栈**: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "PingFang SC", "Microsoft YaHei", sans-serif
- **层级**:
    - 人格代号: 40px, Bold, Letter-spacing: -1px
    - 页面标题: 24px, Medium
    - 正文: 16px, Regular, Line-height: 1.6, Color: #1D1D1F
    - 标签/注释: 12px, Regular, Letter-spacing: 0.5px, Color: #4A4A4C

## 3. 页面详细设计

### 3.1 活动落地页 (Landing Page)
- **顶部**: 满宽头图，高度约 40vh (实图：https://s41.ax1x.com/2026/04/29/peIqDnU.jpg)。
- **背景**: 底部使用 `#2586EB` 与头图底部无缝衔接，向下过渡至 `#F0F7FF`。
- **内容**: 
    - Slogan: 居中大字，使用白色或极高对比度深色，视背景而定。
    - 说明文案: 模块化卡片封装，采用玻璃拟态效果 (Background: rgba(255,255,255,0.8), Backdrop-filter: blur(10px))。
- **底部**: 固定悬浮区域，背景色设为带有透明度的正文背景色。

### 3.2 答题页 (Question Page)
- **进度条**: 顶部极简线形，填充色使用 `#5FA2FF`。
- **题干**: 位于页面中上部，大字呈现。
- **选项**: 垂直排列，卡片形式，具备触感反馈，背景使用纯白。

### 3.3 结果计算过渡页 (Transition Page)
- **核心**: 旋转的足球 DNA 序列，颜色延续 `#2E7DEB`。
- **文案**: “正在扫描你的球迷基因...”、“正在匹配 2026 赛事向量...”。

### 3.4 结果页 (Result Page)
- **人格卡片**: 采用类似名片的圆角卡片布局。
- **插画区**: 延续头图的 3D 柔光风格。
- **雷达图**: 8 维数据可视化，展示性格分布，主色调 `#2E7DEB`。

### 3.5 分享海报 (Poster)
- **布局**: 纵向长图 (750x1334)。
- **包含**: 顶部人格头衔、中部雷达图、底部二维码、直播吧 Logo。

## 4. 异常状态处理
- **网络异常**: 断网插画 + `#2E7DEB` 主色重试按钮。
