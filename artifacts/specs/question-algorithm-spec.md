# Question & Algorithm Spec - WFTI 世界杯球迷人格大鉴赏

## 1. 评分规则
- 16 题固定顺序单选题
- 每题 A / B / C 对应 1 / 2 / 3 分
- 8 个维度由相邻两题聚合：
  - F1 = Q1 + Q2
  - F2 = Q3 + Q4
  - E1 = Q5 + Q6
  - E2 = Q7 + Q8
  - S1 = Q9 + Q10
  - S2 = Q11 + Q12
  - A1 = Q13 + Q14
  - A2 = Q15 + Q16
- 每维度总分范围为 2 至 6 分
- 等级映射：
  - 2~3 => L => 1
  - 4 => M => 2
  - 5~6 => H => 3

## 2. 隐藏人格优先检测
### 2.1 TROL / 杠精球评家
- 触发条件：Q3=A 且 Q12=C 且 Q9=A
- 优先级：高于所有常规人格
- 文案：我不是故意抬杠，我只是觉得这个世界的观点太单一了。如果不跟我过两招，你怎么知道自己的热爱有多深？我毕生的追求，就是在对攻中寻求真理。

### 2.2 HERO / 四年一醒
- 触发条件：Q13=A 且 Q16=C 且 Q7=C
- 优先级：低于 TROL，高于所有常规人格
- 文案：平时我是个路人甲，世界杯开始的那一刻，我能比任何死忠都更疯狂。这种四年一次的觉醒，懂的人自然懂。我不是在追星，我是在寻找那个丢失的自己。

### 2.3 优先级规则
- 若同时满足 TROL 与 HERO，返回 TROL
- 只要命中隐藏人格，即跳过常规人格匹配

## 6. 题库明细
### 题 1
- 维度：F1 忠诚度
- 题干：世界杯分组出来了，你支持的球队分到了「死亡之组」，你的第一反应是？
- A：死亡之组又怎样？八进四我都想好怎么走了，信仰不灭
- B：有点慌，但还是会看，不行就当见证历史了
- C：赶紧看看其他组有没有更值得追的队伍，人生苦短何必吊死一棵树

### 题 2
- 维度：F1 忠诚度
- 题干：你在直播吧看到「你的主队核心球星转会到死敌球队」的新闻，你？
- A：心碎但立场不变，我支持的是球衣上的队徽，不是某个人
- B：纠结好一阵子，可能会同时关注两支队，但主队还是主队
- C：那我也跟着关注新东家吧，球星在哪精彩就在哪

### 题 3
- 维度：F2 专业度
- 题干：直播吧社区有人发帖说「越位规则根本不合理应该取消」，你？
- A：洋洋洒洒写了800字回帖，从造越位战术讲到越位规则的百年演变史
- B：简单回复「越位存在有它的道理」，不太想深入辩论
- C：越位是什么？就是那个让我刚才的欢呼白喊的东西吧？

### 题 4
- 维度：F2 专业度
- 题干：世界杯期间你打开直播吧最先看的内容是？
- A：赛后技术统计+热力图+跑动距离，复盘每个进球的战术逻辑
- B：比分结果和进球集锦，顺便扫一眼新闻评论区
- C：热搜段子和球星表情包，看看今天又有什么名场面可以存图

### 题 5
- 维度：E1 情绪烈度
- 题干：你支持的球队在第94分钟头球绝杀了！你的反应是？
- A：内心波涛汹涌但表面只是用力握了一下拳，然后冷静地看 VAR 回放
- B：猛拍沙发大喊一声，立刻打开直播吧发一条庆祝评论
- C：原地弹射起飞把啤酒泼了一地，抱住旁边的人不管认不认识，邻居以为发生了地震

### 题 6
- 维度：E1 情绪烈度
- 题干：你在直播吧看文字直播，看到一条「VAR 取消进球」的推送，你？
- A：叹口气，打开回放看看到底是不是越位
- B：在评论区发了一串省略号和一个裂开的表情
- C：手机差点没摔出去，连发三条评论骂 VAR 是足球之敌，手在抖

### 题 7
- 维度：E2 抗挫韧性
- 题干：你支持的球队连输两场小组赛，最后一轮必须大比分赢才有理论出线机会，你？
- A：做好心理准备提前进入吃瓜模式，不抱不切实际的幻想
- B：嘴上说「没戏了」但还是默默定了闹钟，万一呢
- C：穿上球衣倒好啤酒，「数学上还没淘汰信仰就不熄灭」，发一条动态说今晚见证奇迹

### 题 8
- 维度：E2 抗挫韧性
- 题干：世界杯决赛你支持的球队输了，赛后你在直播吧的状态是？
- A：伤心但理性接受，发了一句「四年后再来」关掉了 APP
- B：反复刷赛后分析和球员采访，心情低落好几天但慢慢恢复
- C：连续一周在社区发情感帖，看到决赛集锦就破防，已经在倒计时下一届了

### 题 9
- 维度：S1 社交偏好
- 题干：你支持的球队半决赛绝杀晋级了！比赛结束那一刻你最想做的事是？
- A：一个人在黑暗中静静坐了五分钟消化这种快乐，然后默默关掉手机去睡觉
- B：立刻打电话给那个和你一样支持这队的朋友，两人在电话里疯狂复盘每一分钟
- C：冲下楼找到最近的夜宵摊或便利店，跟任何一个还醒着的人分享这个消息，不管认不认识

### 题 10
- 维度：S1 社交偏好
- 题干：你在看球的时候旁边坐了一个陌生人，TA穿着对手球队的球衣，你？
- A：互不打扰，各看各的，我甚至会稍微挪远一点避免尴尬
- B：保持礼貌距离，精彩时刻对视一笑，赛后可能握个手说声「好球」
- C：开场前就开始互相调侃垃圾话，进球了对着TA做庆祝动作，赛后加了微信约下次继续

### 题 11
- 维度：S2 表达欲望
- 题干：世界杯期间你在直播吧社区的活跃程度是？
- A：潜水看帖为主，偶尔点几个赞，不太发言
- B：碰到精彩比赛会发个短评或转发一条新闻，但不是每场都发
- C：赛前预测帖+比赛实时吐槽+赛后复盘长文，评论区出现频率比解说员还高

### 题 12
- 维度：S2 表达欲望
- 题干：有人在直播吧评论区说了一个离谱的观点（比如「梅西不配进世界杯最佳阵容」），你？
- A：看到了但懒得回，各有各的看法
- B：点了一下反对，或者简短回复一句「不敢苟同」
- C：直接开麦长文回击，配图配数据配历史战绩，这事不辩清楚今天不睡了

### 题 13
- 维度：A1 投入方式
- 题干：世界杯即将开始，你的备战状态是？
- A：知道有世界杯就行，有空看看直播，忙了看集锦也可以
- B：重点场次已经标记了，愿意为此调整一下作息
- C：假已经请好了，零食囤完了，赛程表打印贴床头，直播吧通知全开，这个月我只属于足球

### 题 14
- 维度：A1 投入方式
- 题干：直播吧上架了一款世界杯限定周边（球衣/纪念品/实体球星卡），你？
- A：看看热闹不花钱，精神支持即可
- B：喜欢的球队有好看的周边可能会入一件
- C：球衣球鞋围巾贴纸一套拿下，限定版不抢到手心不安，柜子里已经没有位置了

### 题 15
- 维度：A2 冒险倾向
- 题干：直播吧的世界杯竞猜活动开始了，你的策略是？
- A：跟着热门选强队赢，不折腾
- B：大部分跟大众，但偶尔根据自己的分析猜一两场冷门
- C：专挑黑马和大冷门下手，“非洲球队进四强”这种预测我每届都有，万一呢

### 题 16
- 维度：A2 冒险倾向
- 题干：凌晨3点有一场不涉及你主队的「神仙打架」对决（巴西 vs 阿根廷），但你明天有重要事，你？
- A：理性胜出，定好闹钟明天看直播吧的回放和集锦
- B：先睡到凌晨，闹钟响了如果醒了就看，太困就算了
- C：还用想？这是一辈子可能只看一次的对决！看完再说，打哈欠是明天的事
